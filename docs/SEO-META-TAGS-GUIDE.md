# SEO Meta Tags Guide
## Improve Your Portfolio's Search & Social Visibility

---

## Why This Matters

When a hiring manager searches for you or shares your portfolio link on Slack/LinkedIn, these meta tags control:
- **What Google shows** in search results
- **The preview card** that appears when your link is shared
- **How professional** your portfolio appears

---

## Meta Tags for Each Page

Copy and paste these into each page's `<head>` section, right after your GA4 code.

---

### index.html (Portfolio Home)

```html
<!-- Primary Meta Tags -->
<title>Christie Saavedra | UX Designer & Healthcare Technology Specialist</title>
<meta name="title" content="Christie Saavedra | UX Designer & Healthcare Technology Specialist">
<meta name="description" content="UX Designer and Developer specializing in healthcare technology, instructional design, and full-stack development. View my portfolio of healthcare dashboards, learning platforms, and enterprise applications.">
<meta name="keywords" content="UX Designer, Healthcare Technology, Instructional Design, React Developer, Portfolio">
<meta name="author" content="Christie Saavedra">

<!-- Open Graph / Facebook / LinkedIn -->
<meta property="og:type" content="website">
<meta property="og:url" content="https://christiesaavedra.com/">
<meta property="og:title" content="Christie Saavedra | UX Designer & Healthcare Technology Specialist">
```

## Creating OG Images

The `og:image` tags reference preview images that appear when your links are shared. You'll need to create:

| File | Size | Content |
|------|------|---------|
| `og-image.png` | 1200x630px | Portfolio hero with your name/title |
| `og-dashboard.png` | 1200x630px | Dashboard screenshot with title overlay |
| `og-xacteched.png` | 1200x630px | XactechED visual with title |
| `og-arizona.png` | 1200x630px | Arizona Esthetics visual |
| `og-palmview.png` | 1200x630px | Palmview Lab visual |

**Quick option:** Use [Canva](https://canva.com) or [Figma](https://figma.com) to create these. Templates for "Social Media Preview" are 1200x630px.

## Google Search Console Setup

After adding the sitemap, register your site with Google:

1. Go to [search.google.com/search-console](https://search.google.com/search-console)
2. Add property → Enter `https://christiesaavedra.com`
3. Verify ownership (HTML file or DNS method)
4. Submit your sitemap: `https://christiesaavedra.com/sitemap.xml`
5. Wait 1-2 weeks for indexing

---

## Testing Your Meta Tags

Use these tools to preview how your links will appear:

- **LinkedIn Post Inspector:** [linkedin.com/post-inspector](https://www.linkedin.com/post-inspector/)
- **Twitter Card Validator:** [cards-dev.twitter.com/validator](https://cards-dev.twitter.com/validator)
- **Facebook Debugger:** [developers.facebook.com/tools/debug](https://developers.facebook.com/tools/debug/)
